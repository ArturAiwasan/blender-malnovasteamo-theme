## Instalado  

* Ekkopi enhavon de arkivo  
* Malfermi agordojn de Blendero (`Redakti` > `Agordoj`)  
* Transpasi al `Etosoj` sekcio  
* Premi `Instali` butonon kaj elekti *.xml dosieron
